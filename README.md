# Virtualenv


Install

pip install virtualenv

pip install virtualenvwrapper-win

Create virtualenv

Use virtualenv

virtualenv testspy

Use mkvirtualenv will create in C:\Users\{user}\Envs



Activate Windows

activate.bat

or

.\Scripts\activate

Deactivate Windows

deactivate.bat

or

.\Scripts\deactivate.bat
